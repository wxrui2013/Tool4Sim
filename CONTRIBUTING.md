# 贡献指南

感谢您对这个有用Python代码库的兴趣！我们欢迎各种形式的贡献。

## 如何贡献

1. Fork 这个仓库
2. 创建您的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交您的更改 (`git commit -m 'Add some amazing feature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启一个 Pull Request

## 代码规范

- 请确保您的代码有适当的注释
- 遵循PEP 8 Python代码规范
- 添加示例代码以演示用法
- 包含测试代码
- 使用类型提示（如果适用）

## 提交代码时的注意事项

- 确保代码是原创的或具有合适的许可证
- 提供清晰的文档说明代码的用途和用法
- 将代码放在正确的目录中
- 遵循现有的文件命名约定
- 确保所有测试通过

## 问题报告

如果您发现了 bug 或有改进建议，请开启一个 issue 来讨论它。
